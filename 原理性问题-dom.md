# 正则表达式

#### 1. * + 区别
```
正则表达式里：
？出现0次或1次.  [0,1]
* 出现0次或多次.  [0,+∞]
+ 出现1次或多次  [1,+∞]
```

#### 2. Dom树结构
```
DOM树中总共分为如下几种节点格式：Element类型（元素节点）、Text类型（文本节点）、Comment类型（注释节点）、Document类型（document节点）。
第一题：document可以说是一种节点格式，但节点树的根节点也叫document，所以第一题的说法太绝对，是错的。
第二题：所有的HTML elements（元素节点，其实就是HTML标签）都是element。
第三题：comments属于注释节点

```
